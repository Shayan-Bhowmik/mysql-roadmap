## Beginner SQL Interview Questions

### 1. What is SQL?
SQL (Structured Query Language) is used to store, retrieve, and manipulate data in relational databases.

---

### 2. What is a database?
A database is an organized collection of structured data stored electronically.

---

### 3. What is a table?
A table stores data in rows and columns inside a database.

---

### 4. What is the SELECT statement?
SELECT is used to retrieve data from a table.

Example:
SELECT * FROM employees;

---

### 5. What is the WHERE clause used for?
WHERE is used to filter rows based on conditions.

---

### 6. Difference between WHERE and HAVING?
WHERE filters rows before grouping.
HAVING filters groups after GROUP BY.

---

### 7. What is a primary key?
A primary key uniquely identifies each row in a table and cannot be NULL.

---

### 8. What is a foreign key?
A foreign key creates a relationship between two tables.

---

### 9. What is NULL?
NULL represents missing or unknown data.

---

### 10. What is an aggregate function?
Aggregate functions perform calculations on multiple rows and return a single value.
Examples include COUNT, SUM, and AVG.
