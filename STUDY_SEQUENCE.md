# Study Sequence for MySQL Roadmap

This file explains the recommended order to study the files and folders in this repository.

Follow this sequence step by step. Do not skip modules if you are a beginner.

---

## Module 1: Basics

Goal: Understand what SQL is and how to work with simple queries.

Study in this exact order:

1. 01_basics/intro.md  
2. 01_basics/create_database_and_tables.md  
3. 01_basics/select.md  
4. 01_basics/where.md  
5. 01_basics/practice.sql  

Do not move forward until you are comfortable with SELECT and WHERE.

---

## Module 2: Filtering and Sorting

Goal: Learn how to control and organize query results.

Study in this order:

1. 02_filtering_sorting/order_by.md  
2. 02_filtering_sorting/limit_offset.md  
3. 02_filtering_sorting/practice.sql  

Practice these queries multiple times.

---

## Module 3: Aggregations

Goal: Learn how to analyze data using calculations.

Study in this order:

1. 03_aggregations/count_sum_avg.md  
2. 03_aggregations/group_by.md  
3. 03_aggregations/having.md  
4. 03_aggregations/practice.sql  

This module is very important for interviews.

---

## Module 4: JOINs

Goal: Learn how to combine data from multiple tables.

Before starting, ensure you have loaded:
- datasets/employees.sql
- datasets/departments.sql

Study in this order:

1. 04_joins/inner_join.md  
2. 04_joins/left_right_join.md  
3. 04_joins/full_join.md  
4. 04_joins/practice.sql  

JOINs are one of the most important SQL topics.

---

## Module 5: Subqueries

Goal: Learn how to solve complex problems step by step.

Study in this order:

1. 05_subqueries/subqueries.md  
2. 05_subqueries/practice.sql  

Understand when to use subqueries versus JOINs.

---

## Module 6: Constraints and Indexes

Goal: Learn database design and performance basics.

Study in this order:

1. 06_constraints_indexes/constraints.md  
2. 06_constraints_indexes/indexes.md  

This module focuses more on concepts than practice.

---

## Module 7: Real-World SQL Queries

Goal: Apply SQL to realistic business scenarios.

Study in this order:

1. 07_real_world_queries/ecommerce.sql  
2. 07_real_world_queries/company.sql  

Try to understand the logic behind each query.

---

## Module 8: Interview Preparation

Goal: Prepare for SQL interviews and screenings.

Study in this order:

1. 08_interview_questions/beginner.md  
2. 08_interview_questions/intermediate.md  
3. 08_interview_questions/advanced.md  

Use this module for revision and confidence building.

---

## Final Advice

- Always run queries yourself
- Modify queries and experiment
- Focus on understanding, not memorization
- Revisit earlier modules when needed

If you complete all modules in order, you will have a strong foundation in MySQL.
