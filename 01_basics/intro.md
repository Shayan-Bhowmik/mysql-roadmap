## Introduction to SQL and MySQL

### What is SQL?
SQL (Structured Query Language) is a language used to store, retrieve, and manipulate data in relational databases.

It allows you to:
- Read data from databases
- Insert new data
- Update existing data
- Delete data
- Manage database structure

SQL is used in almost every software system that works with data.

---

### What is MySQL?
MySQL is a popular open-source relational database management system that uses SQL as its query language.

It is widely used in:
- Web applications
- Backend systems
- Enterprise software
- Data-driven platforms

In this roadmap, all examples are written using **MySQL syntax**.
